About Platform

Welcome to Online Automated Lecture Schedule Generator. This is a web-based platform which mainly provides a service you to generate lecture schedules for the lecturers instantly without wasting more time like creating them manually. And this allows you to save generated lecture schedules for future or download instantly for present use.

You can find the options in the left side panel for manage users, subjects, lecture halls and also customize your profile info.

NOTE: Currently this is available for Department of Computing & Information Systems only.


Tips & Tricks

Side panel options :

  • Dashboard : Displays summerized view of the system data.

  • Schedule : In there, users can generate lecture schedule instantly within one click. Then users have been facilitated to save or/and download generated lecture schedules.

  • Manage [Only for Administrators]

    • Lecture Halls : This section allows administrators to add, edit and remove lecture halls as well as batch details.
    • Subjects : This section allows administrators to add new subjects, edit and remove added subjects.
    • Users : This section allows administrators to add new users, edit and remove registered users in the system.

  • My Account : This is the profile section which facilitates to update user's personal info. Additionaly this section provides assigned subjects and user's lecture schedule if the user is a lecturer.

  • Help : This section contains the system details and help tips.

How to generate a lecture schedule?

Tap on the Schedule link in left-side panel. Then select the "lecturer name" and the "semester" from the select boxes and press Generate button. Generated lecture schedule displays as a table below to the top section in same Schedule page.

How to save generated lecture schedule?

When lecture schedule generated, There is a button at the top right corner of the displayed lecture shedule section.

  • Click button to save the generated lecture schedule.

How can I find my current lecture schedule?

Tap on the My Account link on left-side panel. Scroll to the bottom of the page and click button under Current Lecture Schedule section in left-side bottom corner. Then it will show a pop-up window with your current lecture schedule as a table.
At the botton left and right corners of the pop-up window has download and delete buttons for download and delete the lecture schedule respectively.

How can I upload a new image for profile picture?

Tap on the My Account link on left-side panel. Click on the square shape profile image on the page. Select a photo which you willing to upload as your profile picture from opened window and click Open. Then click Save Changes button on the page to save selected profile photo. [NOTE : Upload a clear, yourself in the middle image as the profile picture.]

How to change my current password?

Tap on the My Account link on left-side panel. Then click on the Show more options section. It will appear a section with the options for change the current password.
